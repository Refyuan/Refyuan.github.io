Daily Life

2024-03-22
2024-3-22

大学上课，影响学习。我是不明白为什么大三下了给排这么多课，一周四节早八，甚至有一天满课。物联网这专业挂在计算机学院，说是计算机类但是没计算机认可度高，课还比计算机多。

很惶恐，因为比赛和考研焦虑。又开始觉得自己哪儿哪儿都不好了，我又不行了，不配拥有爱，关怀，和他人的信任。用流行的话来说就是有很强的不配得感。自我调理的结果是觉得自己努力少了，把时间都用在卷上就不会胡思乱想了。

挪瓦的冰蓝椰子水好难喝，店员态度也不怎么样，拉黑。

2024-3-23

寒假期间家里人带我理发，我那时不想剪，象征性要求剪短5cm。家里人说再剪些吧，还是太长了。

我说我不觉得长，我习惯这么长了。

听罢理发师露出神秘莫测的微笑，肯定地说不是这个原因。

疑惑，他不会以为我是受什么情伤还是为了某个人才不剪的吧。

然后我返校的第一个周末就嫌洗头麻烦，把头发剪了。从长发变成蘑菇头，去了三分之二。

室友看到我剪短了很惊诧，问我不心疼吗。我笑说不心疼呀。

剪完几周了，感想就是短发洗头很爽，再短点儿应该更爽。

2024-3-25

破案了，是睡眠不足导致的。午睡前边看书边恨这个世界，睡了半小时起来觉得自己又是幸福小孩了，充满干劲。

晚上骑行了1h，心态越来越平和。以前总是想得太多而做得太少。我相信只要开始做了，那么一切都会好的。

2024-3-26

可供了解的情报：

酥饼里脊关门时间在20：00~20：50。

每天抽点儿时间补眠/骑行会让自己心态放平。

事情多了无可避免非常想摄入高热量炸物，这样不好。

目前姑且认为md格式无法不借助插件实现起段落前空两格。等比赛告一段落再美化吧。

2024-3-27

这几天在学数据结构与算法中的“链表”。

起因是看王道考研资料，我连线性表里的指针都半懂不懂，于是查查搜搜，指针与链表相辅相成，然后两天就刷了两道题。

每次自己上手写时都要掰扯半天逻辑关系。不过写完之后发现自己又进步了，好事。

2024-3-31

我把自己的抑郁情绪归结为想得太多做得太少，焦虑情绪归结为车骑少了。

2024-4-2

今日骑车溜进了隔壁。因为没扫码保安大叔想拦我，但是我骑太快了没拦住。自己因为太心虚了骑到南门就出来了。

沿着小吃街溜达，有好朋友问我星穹铁道错误代码，我指指点点了一会儿结果啥忙也没帮上。

与此同时另外一边，一位软工同学告诉我她正在准备预推免面试，结束了就把资料都给我。

有明显的落差感，感觉自己什么都做不到，而身边的大伙都在走向更光明的未来。

提溜了四袋海河奶回来了。好像只有考研上岸才能弥补大学的遗憾了。

2024-4-3

晨起看朋友圈，第一条就是有人上岸复旦了。我评论了恭喜，然后决定完全关闭朋友圈。

恭喜是真的恭喜，年纪渐长，见到大伙的成功只会开心赞叹“好棒，这是你们应得的”，激不起一丝嫉妒的情绪。

只是自己这关好难过呀。脑子里有个声音充当了家长的角色。

你看小a，你们大一时候不是很熟吗，他上岸复旦了你呢？

你看小b，你们一起出去玩，她打比赛做开发，保研就业两手抓，你呢？

我呢？我想说我不知道，但实际上我知道。

我卷了一年绩点，打了两年游戏，一手促成自己玩没玩好，学也没学好的现状。

作为学计算机的，连数据结构都不会，比跨考的还零基础。

作为游戏玩家，好像还行？但单机游戏不能当饭吃啊。

up主确实可以作为副业，不过没人会喜欢满口脏话的吧，乐。

我是客观意义上的互联网鼠鼠，看着成功的大伙，自己阴暗地缩进没有光的下水道。

请阳光的你们不要想起我。我没有勇气面对自己的落差自己的失败。高中的困境我始终没有逃脱。

今时与往日稍有的不同就是，我愿意稍微努力一点，就一点点。

大伙的光照得那样恣意，无人在意角落里鼠鼠的毛发是否比从前光亮。

没关系。鼠鼠自己会在意的。

2024-4-4

他人之得不必视为自己之失。我告诉自己。

如果有人类因为所谓能力在一部分人类面前卑躬屈膝，那ta一定会在另一部分人类面前趾高气扬。

前段时间的更新是逼着自己抱着书和博客硬啃，确实有进步，但是自己心态很崩。

这个算法的概念我明白了，但是怎么实现呢？

def search(self,question) -> answer:
    if question in blog:
        return blog(answer)
    elif question in book:
        return book(answer)
    else:
        ask ChatGPT
        return self.search(new question from ChatGPT)

敲的傻B伪代码给自己看乐了。乐。

人与计算机的不同在于，人会抱怨。

解法没问题的情况下计算机会忠诚地执行每一步，耐心地回溯直到解决。

我自己则是回溯到上一层就开始烦了，开始：

淦哦为什么这么多不知道的事儿——为什么查了这么多依然没有解决当下的问题——烦内！

今天在b站找到了一位讲得不错的算法入门up：灵茶山艾府。

感慨师傅领进门还是重要的，拨云见雾了。

看视频就：なるほど！非常简单，我简直就是天才！（雾）

再敲几行，爽学！

2024-4-6

葬送的芙莉莲为什么是神ヾ(´ωﾟ｀)

2024-4-8

我跟学长说我很难过，因为学了半个多月了，练习一下篮球杯真题还是抓瞎。

力扣只用写方法，但蓝桥要写完整的代码。

我甚至不会处理题目传入的多行不同类型的数据。

只知道input()，然后呢？

不会处理题目传入的数据，一分也拿不到。

到头来还是啥也不会。

学长说，这半个多月我的成长就是从能力排行70%的人类变成了能力排行50%的人类，足足进步20%。

也是，刚开始做博客的时候我只会print(“Hello World”)呢，现在已经在dp缓慢爬行了。

我仅有的优势应该就是我能很快理解算法。

与之相对的就是我只能非常慢地敲出bug堆起来的代码。

循环写得很丑陋。 总要在各种边界条件、角标处修改。

力扣上每次都要增删改查七八遍，质疑编译器三四次。

对写代码的不熟练使我痛苦。

我喜欢看完一个题就理解一种算法技巧的正反馈，

我讨厌debug，看到标红我会气得乳腺增生，但学数据结构与算法就离不开练习，练习离不开debug。

压力好大。

2024-4-9

恢复了阿宅生活，每天在灵茶山艾府和葬送的芙莉莲之间反复横跳。

灵茶山艾府算法讲得很棒，我要是能像他一样行云流水地将算法转化为代码就好了。

葬送的芙莉莲绝对是我这几年看过最棒的番了。一定要补漫画！

可能是年纪大了，见得多了，初高中生的爱恋越看越幼稚，越看越不顺眼。

室友分享了初中生亲妹妹的抖音视频，配图是两张不同字迹的纸，文案是某某永远爱某某。

室友觉得是可爱的乐子，我说我觉得太傻逼了。

“他们口中的 诗呀 爱呀 矫揉太过，难道要我 一起感动吗？”

懵懂稚嫩的情窦初开，我已并不觉得美好。

反复无常，人性如此。多的是变化，多的是不负责任的伤害。

不懂事时期留下多余的情感痕迹只会让后来者难过。

我不知道六花知道七宫的存在后的真实心理，我只知道我被这种重逢设定恶心。

各种gal改我也不知道我当年是怎么看下去的。

还有典中典的73和赤坂两个老贼，为了恰钱硬搞ntr，现在想起来还觉得反胃。

老登纯爱战士很容易被辛美尔和芙莉莲这样的纯爱感动。

ta们本身都是善良成熟的人，有独立人格且能力强大，不存在谁依赖谁。

一起冒险十年，二位是队友是战友更是知己。

我并不觉得阴阳两隔算be，毕竟辛努力让芙记住，芙因为辛再次踏上旅途。

算是在讲成年人的故事吧。

初高的纯爱剧情，我总开始质疑：然后呢？

你们这么纯爱，现在一定还在一起吧？

标榜纯爱的某某同学系列，你们毕业之后呢？

按小日子国情来看，双方压力都很大。时间久了以后：

男方工作应酬不顾家，回家看吵人的老婆小孩不如去红灯区爽爽。

女方烦做家务带小孩，老公不在家一个人寂寞，找别的男人爽爽。

双方：你一点都不体谅我的辛苦！背地里找好友吐槽：对方什么时候似啊。

现在的我只会想到花束般的恋爱，和上一辈人糟糕的婚姻。

爱到底是什么呢。你们懂个p的爱啊，痒了就拿拖鞋拍拍。

我又懂个p的爱啊。
